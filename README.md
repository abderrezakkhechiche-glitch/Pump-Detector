# 🚀 Pump Detector Bot

بوت ذكي لالتقاط البامبات (الارتفاعات المفاجئة) على Binance قبل حدوثها.

## 🎯 المميزات

- كشف انفجار حجم التداول (Volume Spike)
- تحليل RSI و Bollinger Bands
- تحليل أوامر الشراء (Order Book)
- نظام نقاط (0-100) لتحديد قوة الإشارة
- متابعة البامبات السابقة وإبلاغ النتيجة
- يعمل كل 5 دقائق

## 📊 مستويات الثقة

- 70-100: 🔴 ثقة عالية جداً
- 50-69: 🟠 ثقة متوسطة
- 30-49: 🟡 ثقة منخفضة
